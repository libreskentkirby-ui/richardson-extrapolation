Richardson's Extrapolation Module

01. Mathematical Discussion

Richardson's Extrapolation is a sequence acceleration method used to improve the rate of convergence of a numerical approximation. It combines two linear approximations of a specific step size (typically \(h\) and \(\frac{h}{2}\)) to cancel out the leading error terms, thereby yielding a higher-order accurate formula.

When applied to numerical differentiation using the Central Difference scheme of order \(\mathcal{O}(h^2)\), the extrapolated formula of order \(\mathcal{O}(h^4)\) is expressed as:

$$D = \frac{4D\left(\frac{h}{2}\right) - D(h)}{3}$$

Where \(D(h)\) and \(D\left(\frac{h}{2}\right)\) are the standard central difference estimates for step sizes \(h\) and \(\frac{h}{2}\) respectively:

$$D(h) = \frac{f(x + h) - f(x - h)}{2h}$$

02. Fully Worked Step-by-Step Examples

Example 1: \(f(x) = x^2\) at \(x = 2.0\)

Config: Step size \(h = 0.1\). Exact target derivative is \(4.0\).

Step 1: Compute \(D(0.1)\)

  • \(f(2.1) = 2.1^2 = 4.41\)
  • \(f(1.9) = 1.9^2 = 3.61\)
  • $$D(0.1) = \frac{4.41 - 3.61}{0.2} = 4.0000$$

Step 2: Compute \(D(0.05)\)

  • \(f(2.05) = 2.05^2 = 4.2025\)
  • \(f(1.95) = 1.95^2 = 3.8025\)
  • $$D(0.05) = \frac{4.2025 - 3.8025}{0.1} = 4.0000$$

Step 3: Extrapolate

$$D = \frac{4(4.0000) - 4.0000}{3} = 4.0000$$

Example 2: \(f(x) = x^3\) at \(x = 1.0\)

Config: Step size \(h = 0.2\). Exact target derivative is \(3.0\).

Step 1: Compute \(D(0.2)\)

  • \(f(1.2) = 1.2^3 = 1.728\)
  • \(f(0.8) = 0.8^3 = 0.512\)
  • $$D(0.2) = \frac{1.728 - 0.512}{0.4} = 3.0400$$

Step 2: Compute \(D(0.1)\)

  • \(f(1.1) = 1.1^3 = 1.331\)
  • \(f(0.9) = 0.9^3 = 0.729\)
  • $$D(0.1) = \frac{1.331 - 0.729}{0.2} = 3.0100$$

Step 3: Extrapolate

$$D = \frac{4(3.0100) - 3.0400}{3} = 3.0000$$
